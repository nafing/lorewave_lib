# Chromium 103 compatibility

This workspace does not contain native CEF or C++ integration code.
The adaptation in this repository covers the browser-facing UI bundle that is meant to run inside an embedded Chromium runtime such as FiveM NUI / CEF.

## What was changed

- Vite build target was pinned to `chrome103`.
- CSS target was pinned to `chrome103`.
- App TypeScript target and libs were reduced from `ES2023` to `ES2021` so editor hints and future code changes stay closer to the actual browser baseline.

## Notes

- No native `CefBrowser`, `CefClient`, IPC, render-process, or V8 handler code exists in this repository.
- Current source code does not appear to rely on browser APIs that are obviously newer than Chromium 103.
- The explicit compatibility guarantee in this repo now applies to the production bundle generated by Vite.
- If you also maintain a separate native CEF host application, its migration still needs to be handled in that codebase.
